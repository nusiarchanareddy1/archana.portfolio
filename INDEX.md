## 📞 Getting Help

### Common Issues:

**Q: Changes not showing?**
A: Hard refresh browser (Ctrl+Shift+R)

**Q: Mobile view looks broken?**
A: Press F12, click device toggle, check sizes

**Q: Links not working?**
A: Check URLs are correct, no typos

**Q: Dark mode not saving?**
A: Clear browser cache, enable localStorage

**Q: Form not working?**
A: It's demo only - add backend if needed
